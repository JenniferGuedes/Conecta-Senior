<h2>Bem Vindos ao Conecta Sênior!</h2>
 <h3>Mas o que seria o Conecta Sênior?</h3> 
<p>O Conecta Sênior é um site criado para o meu segundo projeto de Extensão Curricular da Universidade Santa Cecília - Santos/São Paulo. Seu objetivo é auxiliar o cuidador de idosos consiga fazer uma rotina organizada para o idoso, contando com medicamentos que ele toma, e para organizarem melhor sua rotina de tomada de medicamentos, utilizando o primeiro projeto "Como Tomar?" para adicionarem informações pertinentes ao tratamento do idoso.</p>
<h3>E por qual motivo devo usá-lo?</h3>
<p>Sabem-se que os cuidadores de idosos, muitas vezes, tem mais de um idoso para cuidar, tanto de forma residencial, como trabalhando em casas de repouso, exigindo que o mesmo decore os medicamentos e rotinas de cada um. Porém, acaba se tornando uma tarefa muito mais cansativa que o habitual por estes motivos. Este projeto foi criado para ser um facilitador na rotina dos cuidadores de idosos, afinal, muitos cuidam de idosos em tempo integral, levam ao médico, acompanham em passeios, em farmácias, mercados, enfim, em sua rotina diária. Por este motivo, o projeto do site, agora em estágio inicial de desenvolvimento, conta com uma Ficha, que será feita pelo cuidador, com algum medicamento novo que ele vá tomar, e os medicamentos listados, que ele já faz uso contínuo, para que seja impressa, e fique em fácil acesso do cuidador. Também auxilia quem está começando na profissão, afinal, vai impedir que o novo cuidador se atrapalhe na organização da rotina. </p>
<p>Também tem a opção de acessar o "Como Tomar?", caso o idoso se queixe ou tenha dúvidas sobre o seu tratamento. Lembrando que a ficha de medicamentos do "Como Tomar?" conta com acessibilidade para os idosos que não são alfabetizados, e por este motivo, criam uma resistência maior para tomar medicamentos novos, ou trocar o laboratório do medicamento que já tomam.</p>
